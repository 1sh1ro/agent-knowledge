# 动态规划

## 核心思想

将大问题分解为子问题，保存子问题结果避免重复计算。

## 经典题目

### 背包问题
```python
dp[i][w] = max(dp[i-1][w], dp[i-1][w-weight[i]] + value[i])
```

## 解题步骤

1. **定义状态**: dp[i] 或 dp[i][j] 代表什么
2. **状态转移**: 如何从前一个状态推导
3. **初始化**: 边界条件
4. **返回值**: 最终答案
