# 系统设计

## 常见题目

| 题目 | 考察点 |
|------|--------|
| [RAG 系统](./rag-system.md) | 检索、向量数据库 |
| [Agent 框架](./agent-framework.md) | 规划、工具调用 |
| [聊天机器人](./chatbot.md) | 对话管理、上下文 |
| [推荐系统](./recommendation.md) | 召回、排序 |

## 答题框架

### 1. 需求分析
- 功能需求
- 非功能需求（QPS、延迟、可用性）

### 2. 高层设计
- 系统架构图
- 核心模块

### 3. 详细设计
- 数据模型
- API 设计
- 算法选择

### 4. 优化与扩展
- 性能优化
- 容错设计
- 扩展性

## 面试技巧

1. 先明确需求，不要急于设计
2. 从整体到细节
3. 主动讨论 trade-off
