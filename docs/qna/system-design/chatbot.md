# 设计聊天机器人

## 架构

```
用户输入
  ↓
意图识别（Intent Classification）
  ↓
实体抽取（Entity Extraction）
  ↓
对话状态跟踪（DST）
  ↓
对话策略（Policy）
  ↓
回复生成（NLG）
  ↓
用户
```

## 核心能力

| 能力 | 实现 |
|------|------|
| 多轮对话 | 上下文管理 |
| 知识问答 | RAG |
| 任务执行 | Tool Use |
| 个性化 | 用户画像 |

## 评估指标

- 对话轮次
- 任务完成率
- 用户满意度
- 响应时间

## 面试要点

1. **如何处理多轮对话的上下文？**
   - 滑动窗口、摘要、关键信息提取

2. **如何评估聊天机器人效果？**
   - 人工评估 + 自动指标
