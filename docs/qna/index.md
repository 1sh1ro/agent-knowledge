## 🎭 行为面 (BQ)

### STAR 法则

- **Situation**: 描述背景和情境
- **Task**: 说明你的任务和目标
- **Action**: 你采取了什么行动
- **Result**: 最终结果如何

### 高频问题

| 问题 | 考察点 |
|------|--------|
| 介绍一下你做过的最有挑战的项目 | 项目能力、技术深度 |
| 遇到过最难的技术难题是什么 | 问题解决能力 |
| 你为什么想来我们公司 | 动机与规划 |
| 你最近在学什么 | 学习能力、自我驱动 |

### 回答模板

```markdown
## 项目介绍模板

**项目背景**: 为了解决 XXX 问题
**我的角色**: 作为 XXX，负责 XXX
**技术方案**: 采用了 XXX 技术/架构
**核心挑战**: 遇到了 XXX 困难，通过 XXX 解决
**最终效果**: 实现了 XXX 提升/改进
```
