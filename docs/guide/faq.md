# ❓ 常见问题 FAQ

## 关于面经

### Q: 如何提交面经？
A: 点击 Issues → New Issue → 选择「面经分享」模板，按格式填写即可。

### Q: 面经需要多详细？
A: 建议包含：
- 基本信息（公司、岗位、时间）
- 面试流程（各轮问题）
- 结果（通过/未通过）

### Q: 可以匿名提交吗？
A: 可以，所有提交者信息可选。

## 关于知识库

### Q: 如何补充知识库？
A: Fork 仓库后在对应目录添加 Markdown 文件，提交 PR 即可。

### Q: 知识点有格式要求吗？
A: 建议包含概述、核心概念、常见问题和参考资料。

## 关于面试题库

### Q: 如何添加新题目？
A: 在对应分类目录下创建新文件，提交 PR。

### Q: 需要提供答案吗？
A: 建议提供，但不强制。答案会经过审核。

## 技术问题

### Q: 如何本地预览网站？
A: 
```bash
npm install
npm run docs:dev
```

### Q: 网站托管在哪里？
A: 自动部署到 Vercel/GitHub Pages。
