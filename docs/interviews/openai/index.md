# OpenAI 面经

## 公司介绍

OpenAI 是全球领先的 AI 研究机构，开发了 GPT 系列模型。

## 面试特点

- 注重基础研究能力
- 考察深度学习和 NLP 基础
- 关注研究经历和论文

## 面试流程

1. **简历筛选**
2. **Phone Screen**（1-2轮）
3. **Onsite/Virtual**（4-5轮）
   - Coding
   - ML System Design
   - Research Deep Dive
   - Behavioral

## 准备建议

1. 深入理解 Transformer 架构
2. 熟悉训练大模型的实践经验
3. 准备研究项目的详细讲解
4. 了解 OpenAI 的最新研究方向
