# Multi-Agent 系统

## 基本概念

多个 Agent 协作完成任务。

## 协作模式

| 模式 | 描述 | 示例 |
|------|------|------|
| 分工协作 | 各 Agent 负责不同子任务 | 软件开发团队 |
| 竞争模式 | 多个 Agent 提出方案，择优 | 辩论赛 |
| 层级结构 | 管理者 Agent 分配任务 | 项目经理-开发者 |

## 典型框架

### AutoGen
- 微软开发
- 支持对话式编程
- 可配置人机协作

### CrewAI
- 角色扮演
- 任务分配
- 流程编排

### MetaGPT
- 模拟软件公司
- 不同角色（PM、架构师、开发者）

## 面试要点

1. **Multi-Agent 的优势？**
   - 任务分解、专业化、容错性

2. **通信机制如何设计？**
   - 消息传递、共享状态、黑板模型
