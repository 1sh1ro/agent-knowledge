# Agent 面试知识库

> 收集 Agent 相关岗位面试经验、知识点整理、面试题库和学习资源

## 🎯 项目目标

- 整理真实的 Agent 面试经验
- 系统化梳理 Agent 核心知识点
- 汇总高频面试题与参考答案
- 整理优质学习资源

## 🏷️ 标签体系

### 按公司分类

## 🤝 如何贡献

### 方式一：提交 Issue
点击 [New Issue](https://github.com/YOUR_USERNAME/agent-interview-wiki/issues/new/choose) 选择模板提交面经或问题。

### 方式二：直接提交 PR
1. Fork 本仓库
2. 在对应目录下添加内容
3. 提交 PR 并描述你的贡献

### 方式三：补充知识库
在 `docs/knowledge/` 目录下添加你整理的知识点。
